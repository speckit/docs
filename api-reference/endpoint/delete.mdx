---
title: "Delete User"
api: "DELETE https://api.speckit.dev/v1/users/{id}"
description: "Permanently delete a user account"
---

<Warning>
  This action is irreversible. Once a user is deleted, all associated data will be permanently removed.
</Warning>

## Path Parameters

<ParamField path="id" type="string" required>
  The unique identifier for the user to delete
</ParamField>

## Query Parameters

<ParamField query="force" type="boolean" default="false">
  Force deletion even if the user has active projects or subscriptions
</ParamField>

<ParamField query="transfer_to" type="string">
  User ID to transfer ownership of projects and resources to before deletion
</ParamField>

## Response

<ResponseField name="deleted" type="boolean">
  Confirmation that the user was successfully deleted
</ResponseField>

<ResponseField name="id" type="string">
  The ID of the deleted user
</ResponseField>

<ResponseField name="deleted_at" type="string">
  ISO 8601 timestamp of when the user was deleted
</ResponseField>

<ResponseField name="transferred_resources" type="object">
  Information about resources that were transferred (if `transfer_to` was specified)
  
  <Expandable title="transferred_resources">
    <ResponseField name="projects" type="number">
      Number of projects transferred
    </ResponseField>
    
    <ResponseField name="api_keys" type="number">
      Number of API keys transferred
    </ResponseField>
    
    <ResponseField name="transferred_to" type="string">
      ID of the user who received the transferred resources
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>

```bash cURL
curl -X DELETE \
  'https://api.speckit.dev/v1/users/user_123?force=true' \
  -H 'Authorization: Bearer your-api-key' \
  -H 'Content-Type: application/json'
```

```bash cURL (with transfer)
curl -X DELETE \
  'https://api.speckit.dev/v1/users/user_123?transfer_to=user_456' \
  -H 'Authorization: Bearer your-api-key' \
  -H 'Content-Type: application/json'
```

```python Python
import requests

# Simple deletion
url = "https://api.speckit.dev/v1/users/user_123"
headers = {
    "Authorization": "Bearer your-api-key",
    "Content-Type": "application/json"
}
params = {"force": True}

response = requests.delete(url, headers=headers, params=params)
result = response.json()
print(result)
```

```python Python (with transfer)
import requests

# Deletion with resource transfer
url = "https://api.speckit.dev/v1/users/user_123"
headers = {
    "Authorization": "Bearer your-api-key",
    "Content-Type": "application/json"
}
params = {"transfer_to": "user_456"}

response = requests.delete(url, headers=headers, params=params)
result = response.json()
print(result)
```
