## Basic Code Blocks

Use triple backticks with a language identifier for syntax highlighting:

```javascript
const speckit = {
  name: "Speckit",
  version: "1.0.0",
  features: ["documentation", "api", "analytics"]
};

console.log(speckit);
```

```python
def hello_speckit():
    """A simple function to demonstrate Python syntax highlighting."""
    message = "Hello from Speckit!"
    print(message)
    return message

hello_speckit()
```

```bash
# Install Speckit CLI
npm install -g speckit-cli

# Initialize a new project
speckit init my-project

# Start development server
speckit dev
```

## Interactive Code Examples

<CodeGroup>
```javascript JavaScript
// Initialize Speckit client
import { Speckit } from 'speckit-sdk';

const client = new Speckit({
  apiKey: 'your-api-key'
});

// Make API call
const response = await client.getData();
console.log(response);
```

```python Python
# Initialize Speckit client
from speckit import Speckit

client = Speckit(api_key='your-api-key')

# Make API call
response = client.get_data()
print(response)
```

```curl cURL
curl -X GET \
  https://api.speckit.dev/data \
  -H 'Authorization: Bearer your-api-key' \
  -H 'Content-Type: application/json'
```
</CodeGroup>
